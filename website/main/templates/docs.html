AI Content Predictor

Analyze and predict whether content is AI-generated or human-made.

About

The AI Content Predictor helps analyze different types of content (audio, images, text) to determine if it is AI-generated or created by a human. Upload your files to get instant predictions.

Technologies Used

  • HTML: For structuring the user interface.
  • CSS: For styling and responsive design.
  • Python: Backend language for processing and model integration.
  • Django: Web framework for building and managing the application.
  • TensorFlow/Keras: For deep learning model predictions.
  • Librosa: For audio feature extraction.
  • Joblib: For handling text vectorization and model serialization.
  • OpenCV: For image processing.

Features

  • Predict whether an uploaded audio file is AI-generated or human voice.
  • Analyze text files to determine the likelihood of AI generation.
  • Examine images to identify if they are AI-created or real.

How to Use

Steps

  1. Click the "Upload File" button.
  2. Select an audio, image, or text file from your device.
  3. Click "Submit" to analyze the file.
  4. View the prediction result displayed below the upload form.
